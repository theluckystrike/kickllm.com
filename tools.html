Tools & Resources

Everything you need to start exploring, experimenting with, and building on large language models.

Ollama

The easiest way to run LLMs locally on your computer. One command to download and run models like Llama 3, Mistral, and Phi. Perfect for beginners who want to experiment without paying for API access.

github.com/ollama/ollama

Hugging Face Chat

A free web-based interface for chatting with open-source models. No installation required. Great for quickly comparing different models and getting a feel for their strengths and weaknesses before committing to one.

huggingface.co/chat

LM Studio

A desktop application with a visual interface for downloading, managing, and running local LLMs. Includes a chat interface and local API server. Ideal for users who prefer a graphical interface over command-line tools.

lmstudio.ai

OpenRouter

A single API that provides access to dozens of models from different providers. Try GPT-4, Claude, Gemini, Llama, and many others through one unified interface with pay-per-use pricing and no monthly commitments.

openrouter.ai

Zovo.one

A comprehensive directory of AI tools, platforms, and resources. An excellent starting point for exploring the full AI ecosystem and discovering tools beyond the most well-known options.

zovo.one

Chatbot Arena (LMSYS)

A crowdsourced platform where users compare anonymous LLM responses side by side and vote for the better one. The resulting Elo ranking is widely considered the most reliable measure of real-world model quality.

chat.lmsys.org

Anthropic Prompt Engineering Guide

The official guide to prompt engineering from Anthropic. Covers the fundamentals of writing effective prompts with clear examples and explanations that apply across different LLM providers.

docs.anthropic.com

Tokenizer Playground

OpenAI's interactive tokenizer tool that shows you exactly how text gets split into tokens. Essential for understanding context windows, estimating costs, and learning why certain prompts use more tokens than you might expect.

platform.openai.com/tokenizer

Open LLM Leaderboard

Hugging Face's comprehensive benchmark of open-source language models. Track performance across multiple evaluation categories to make informed model selection decisions for your projects.

huggingface.co — Open LLM Leaderboard
