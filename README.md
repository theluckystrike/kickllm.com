# KickLLM

> Free LLM API cost calculator comparing Claude, GPT-4, Gemini, Llama, and Mistral pricing side-by-side.

**[Use KickLLM live](https://kickllm.com/)**

KickLLM is a free LLM API cost calculator that compares pricing across every major provider — Anthropic Claude, OpenAI GPT-4, Google Gemini, Meta Llama via Groq, and Mistral. Estimate monthly spend, per-conversation costs, and the break-even point where self-hosting beats pay-as-you-go APIs, all with real pricing data for 40+ models.

## Features

- Monthly cost calculator with configurable request volume, input and output token counts
- Per-conversation cost breakdown for multi-turn chat applications
- Self-hosting break-even analysis comparing API spend vs. GPU infrastructure costs
- Real pricing data for 40+ models across Anthropic, OpenAI, Google, Meta, and Mistral
- Interactive comparison table sorted by total cost, quality, or speed
- Input/output token ratio slider for accurate realistic estimates
- LLM value index ranking models by quality per dollar
- Use-case recommendations (coding, summarization, chat, long context)
- 100% client-side — no data leaves your browser
- MIT licensed
- No signup, no tracking
